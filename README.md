# 🌸 Dreamy Frames Gallery
Dreamy Frames✨ — A modern Pinterest-inspired responsive image gallery built using HTML, CSS, and JavaScript,inspired image gallery with dark mode, slideshow, filters, and lightbox effects

✨ Features:
• Responsive Gallery Layout
• Dark / Light Mode Toggle
• Auto Slideshow
• Category Filter Buttons
• Lightbox Image Preview
• Next / Previous Navigation
• Favorite / Like Button ❤️
• Image Zoom Hover Effects
• Smooth Transitions & Animations
• Pinterest-style Aesthetic UI

🛠️ Technologies Used:
• HTML5
• CSS3
• JavaScript
